# Problem Statement
淘宝抢购手速或者网速慢，抢不到

# Description
自动抢购，目标是从高层(浏览器界面层)实现自动刷新点击下单

# Use Cases
设置时间，物品链接后，可实现自动刷新点击下单

# Design Sketches
使用Selenium实现浏览器的自动控制，脚本读取配置文件后，在指定之间实现自动刷新下单功能
